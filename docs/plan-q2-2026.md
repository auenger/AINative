## 技术风险与应对

| 风险 | 影响 | 概率 | 应对措施 |
|------|------|------|----------|
| Tauri V2 API 变更 | 桌面端核心功能阻塞 | 中 | 锁定 Tauri 版本，关注 release notes |
| Monaco 编辑器性能瓶颈 | 大文件编辑体验差 | 中 | 虚拟化渲染 + Web Worker 分离 |
| Agent CLI 桥接兼容性 | Agent 功能不可用 | 高 | 统一事件格式 + 优雅降级 + 重试机制 |
| FS-as-Database 并发安全 | 数据损坏/丢失 | 低 | 文件锁 + 原子写入 + 备份机制 |
| LLM API 配额限制 | Agent 服务不稳定 | 中 | 本地缓存 + 请求队列 + 多 Provider 切换 |
| 多模态模型延迟 | 用户体验差 | 中 | 异步处理 + 进度反馈 + 流式输出 |
