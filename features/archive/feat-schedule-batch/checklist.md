# 验收清单: 批量操作 (feat-schedule-batch)

## 功能验收

### 批量表单
- [ ] 可以批量选择用户
- [ ] 支持按部门筛选用户
- [ ] 可以指定日期范围
- [ ] 可以选择班次类型
- [ ] 可以填写工作地点
- [ ] 可以填写备注
- [ ] 表单验证正常工作

### 预览功能
- [ ] 点击预览显示将要创建的排班
- [ ] 预览显示每个用户每天的排班计划
- [ ] 预览显示总记录数
- [ ] 预览列表清晰易读

### 批量创建
- [ ] 确认后成功创建所有排班记录
- [ ] 自动跳过已有排班的日期
- [ ] 显示批量创建结果
- [ ] 结果包含成功/跳过/失败数量

### 权限控制
- [ ] 管理员可以为所有用户批量排班
- [ ] 部门领导只能为本部门员工批量排班
- [ ] 用户选择器根据角色过滤数据

### 通知功能
- [ ] 批量创建成功后通知所有相关用户
- [ ] 通知内容包含日期范围、班次、地点

### 用户体验
- [ ] 批量表单界面友好
- [ ] 预览功能让用户确认后再创建
- [ ] 创建结果清晰显示
- [ ] 可以取消批量操作
