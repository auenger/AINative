## 开发注意事项

1. **依赖关系**：
   - 必须先完成 `feat-schedule-basic`
   - 复用基础功能的 API 和服务
   - 复用基础功能的表单组件

2. **日历逻辑**：
   - 正确计算每月天数和第一天星期几
   - 处理闰年2月天数
   - 处理跨月显示

3. **性能优化**：
   - 按月加载排班数据
   - 避免一次性加载过多数据

4. **权限控制**：
   - 部门领导只能看本部门数据
   - 管理员可以看所有数据
