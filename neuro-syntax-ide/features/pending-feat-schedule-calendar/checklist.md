# 验收清单: 日历视图 (feat-schedule-calendar)

## 功能验收

### 日历视图
- [ ] 日历正确显示当前月份
- [ ] 正确显示每月天数（包括闰年2月）
- [ ] 正确显示每月第一天是星期几
- [ ] 支持切换到上一月
- [ ] 支持切换到下一月
- [ ] 支持"回到今天"功能

### 排班显示
- [ ] 有排班的日期显示班次颜色标识
- [ ] 颜色正确对应班次类型
- [ ] 无排班的日期显示空白
- [ ] 多个排班的日期正确显示

### 详情弹窗
- [ ] 点击日期弹出详情弹窗
- [ ] 弹窗显示日期信息
- [ ] 弹窗显示当日所有排班
- [ ] 排班信息完整（用户、班次、时间、地点、备注）
- [ ] 弹窗有关闭按钮

### 弹窗操作
- [ ] 支持在弹窗中编辑排班
- [ ] 支持在弹窗中删除排班
- [ ] 支持在弹窗中创建新排班
- [ ] 操作后日历视图正确更新

### 筛选功能
- [ ] 支持按用户筛选
- [ ] 支持按部门筛选
- [ ] 筛选后日历正确更新
- [ ] 可以清除筛选条件

### 权限控制
- [ ] 部门领导只能看本部门数据
- [ ] 管理员可以看所有数据
- [ ] 无权限用户无法访问

### 用户体验
- [ ] 鼠标悬停显示简要信息
- [ ] 点击日期有视觉反馈
- [ ] 弹窗打开/关闭流畅
- [ ] 月份切换有过渡动画
