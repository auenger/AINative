# Spec: 日历视图 (feat-schedule-calendar)

## 📋 需求概述

为排班管理提供日历视图，以月历形式直观展示排班情况，支持月份切换、点击查看详情等功能。

## 🎯 功能需求

### 1. 功能模块

#### 1.1 日历视图
- 以月历形式展示排班情况
- 支持切换月份查看
- 点击日期可查看/编辑当日排班
- 支持按用户/部门筛选
- 班次颜色区分显示

#### 1.2 日历单元格
- 显示当日排班数量
- 显示班次类型（颜色标识）
- 支持鼠标悬停显示简要信息
- 无排班的日期显示空白

#### 1.3 详情弹窗
- 点击日期弹出当日排班详情
- 支持在弹窗中直接编辑/删除排班
- 支持在弹窗中创建新排班

## 🔧 技术方案

### 前端实现

#### 新增页面
- `/schedules` - 排班管理主页（日历视图）

#### 新增组件
- `ScheduleCalendar.tsx` - 日历视图组件
  - 月历布局
  - 月份切换
  - 日期单元格渲染
  - 班次颜色标识
  - 筛选器（用户/部门）

#### 新增服务
- 依赖 `feat-schedule-basic` 的 `scheduleService.ts`

### 样式设计

#### 班次颜色配置
```typescript
const SHIFT_CONFIG = {
  early: { name: '早班', color: '#10b981' },      // 绿色
  morning: { name: '上午班', color: '#3b82f6' },   // 蓝色
  afternoon: { name: '下午班', color: '#f59e0b' }, // 橙色
  night: { name: '晚班', color: '#8b5cf6' }       // 紫色
};
```

#### 日历布局
- 7列（周一到周日）
- 5-6行（根据月份天数）
- 响应式设计，支持移动端

## ✅ 验收标准

### Gherkin Scenarios

```gherkin
Feature: 排班日历视图

  Scenario: 管理员查看日历视图
    Given 管理员登录系统
    And 系统中有排班记录
    When 管理员访问排班管理页面 "/schedules"
    Then 系统显示当前月份的日历视图
    And 有排班的日期显示班次颜色标识
    And 无排班的日期显示空白

  Scenario: 切换月份查看
    Given 管理员在排班日历页面
    And 当前显示"2024年3月"
    When 管理员点击"下一月"按钮
    Then 日历切换到"2024年4月"
    And 显示4月份的排班记录

  Scenario: 点击日期查看排班详情
    Given 管理员在排班日历页面
    And "2024-03-15"有排班记录
    When 管理员点击"2024-03-15"日期单元格
    Then 系统弹出排班详情弹窗
    And 弹窗显示当日所有排班记录
    And 弹窗显示用户、班次、时间、地点信息

  Scenario: 在详情弹窗中编辑排班
    Given 管理员在排班日历页面
    And "2024-03-15"有"张三"的早班排班
    When 管理员点击"2024-03-15"日期单元格
    And 管理员在详情弹窗中点击"张三"排班的"编辑"按钮
    And 管理员将班次改为"晚班"
    And 管理员点击"保存"
    Then 日历视图更新显示
    And "2024-03-15"的班次颜色变为紫色

  Scenario: 在详情弹窗中创建新排班
    Given 管理员在排班日历页面
    And "2024-03-15"没有"李四"的排班
    When 管理员点击"2024-03-15"日期单元格
    And 管理员在详情弹窗中点击"新增排班"
    And 管理员选择用户"李四"
    And 管理员选择班次"早班"
    And 管理员填写工作地点"总部大楼"
    And 管理员点击"保存"
    Then 日历视图更新显示
    And "2024-03-15"显示新增的排班

  Scenario: 按用户筛选日历
    Given 管理员在排班日历页面
    And 系统中有多个用户的排班
    When 管理员在筛选器中选择用户"张三"
    Then 日历只显示"张三"的排班
    And 其他用户的排班不显示

  Scenario: 部门领导查看本部门日历
    Given 部门领导"王经理"登录系统
    And "王经理"是"技术部"的部门领导
    And "技术部"有员工"张三"、"李四"
    When "王经理"访问排班日历页面
    Then 日历只显示"技术部"员工的排班
    And "王经理"可以查看/编辑/删除排班
```
