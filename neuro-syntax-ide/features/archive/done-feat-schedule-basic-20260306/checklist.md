# 验收清单: 基础排班管理 (feat-schedule-basic)

## 功能验收

### 排班查看
- [ ] 普通用户可以查看自己的排班（只读列表）
- [ ] 普通用户只能查看，不能编辑
- [ ] 部门领导可以查看本部门所有员工排班
- [ ] 管理员可以查看所有用户排班
- [ ] 列表视图正确显示排班信息

### 排班创建
- [ ] 管理员可以创建单个排班
- [ ] 部门领导可以为本部门员工创建排班
- [ ] 表单验证正常工作（必填项检查）
- [ ] 选择班次后自动填充默认时间
- [ ] 创建后排班记录正确保存
- [ ] 创建后用户收到通知

### 排班编辑
- [ ] 管理员可以编辑任意排班
- [ ] 部门领导可以编辑本部门排班
- [ ] 编辑后记录正确更新
- [ ] 编辑后用户收到通知

### 排班删除
- [ ] 管理员可以删除任意排班
- [ ] 部门领导可以删除本部门排班
- [ ] 删除前有确认提示
- [ ] 删除后记录正确移除
- [ ] 删除后用户收到通知

### 冲突检测
- [ ] 同一用户同一天创建第二个排班时显示冲突提示
- [ ] 冲突检测阻止重复排班保存
- [ ] 冲突提示信息清晰明确

### 列表筛选
- [ ] 表格正确显示排班记录
- [ ] 支持按日期范围筛选
- [ ] 支持按用户筛选
- [ ] 支持按部门筛选
- [ ] 支持按班次筛选
- [ ] 筛选结果正确更新

### 权限控制
- [ ] 普通用户访问 `/schedules/list` 被拒绝
- [ ] 部门领导只能看到本部门数据
- [ ] 管理员可以看到所有数据
- [ ] 未登录用户被重定向到登录页

### 通知功能
- [ ] 创建排班后用户收到通知
- [ ] 修改排班后用户收到通知
- [ ] 删除排班后用户收到通知
- [ ] 通知内容包含完整信息（日期、班次、地点、备注）
