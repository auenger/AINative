# Feature: 通知历史 (feat-notification-history)

## 概述

用户可以查看自己的历史通知记录，支持按时间、类型、状态筛选和搜索。

## 需求描述

### 功能需求

1. **历史通知列表**
   - 显示用户的所有历史通知
   - 按时间倒序排列
   - 支持分页加载

2. **筛选和搜索**
   - 按通知类型筛选
   - 按状态筛选（已读/未读）
   - 按日期范围筛选
   - 按关键词搜索

3. **批量操作**
   - 批量标记为已读
   - 批量删除

### 用户场景

- **场景 1**：用户查看通知历史
  - Given: 用户已登录
  - When: 进入通知历史页面
  - Then: 显示所有历史通知

- **场景 2**：用户筛选通知
  - Given: 用户在通知历史页面
  - When: 选择筛选条件
  - Then: 显示符合条件的通知

- **场景 3**：用户搜索通知
  - Given: 用户在通知历史页面
  - When: 输入搜索关键词
  - Then: 显示包含关键词的通知

- **场景 4**：用户批量删除
  - Given: 用户选中多条通知
  - When: 点击删除按钮
  - Then: 选中的通知被删除

### 技术方案

```typescript
interface NotificationHistoryFilters {
  type?: string;
  status?: 'read' | 'unread' | 'all';
  date_from?: string;
  date_to?: string;
  keyword?: string;
}
```
